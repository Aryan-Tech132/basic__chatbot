# Basic Chatbot (Python)

A simple rule-based chatbot built using Python. The chatbot responds to predefined user inputs using conditional statements and functions.

## Features

- Responds to basic greetings and questions.
- Uses predefined replies.
- Runs continuously until the user exits.
- Simple command-line interface.
- Beginner-friendly Python project.

## Concepts Used

- If-Elif Statements
- Functions
- Loops
- Input/Output
- Strings

- hello → Hi!
- how are you → I'm fine, thanks!
- bye → Goodbye!
